# Mr Chef

Mr Chef is a recipe browsing app built with Vue.

I made this project to practice working with component structure, filtering, modals, favorites, localStorage, and responsive UI styling in a more polished, product-style layout.

## What it does

Users can:

- browse a collection of recipes
- search by recipe name
- filter by meal type
- filter by cooking time
- filter by minimum rating
- sort recipes
- mark recipes as favorites
- view favorite recipes only
- open a recipe modal with more details
- adjust the serving size and automatically scale ingredient amounts

Favorites are saved with `localStorage`, so they stay there after refresh.

## Built with

- Vue 3
- Composition API
- JavaScript
- CSS
- local recipe data

## Features

- recipe search
