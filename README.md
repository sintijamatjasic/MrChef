## About this project

I built this project as part of my Vue learning process, with the goal of practicing component-based architecture, shared parent state, props, emits, `v-model`, filtering logic, and building a more realistic mini application.

The primary focus here is the Vue implementation and application logic. The visual styling was not designed by me and is included mainly to support presentation, while my attention remained on the coding patterns and frontend concepts the project was meant to reinforce.
